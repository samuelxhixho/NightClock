# NightClock

NightClock is a minimalist OLED-friendly Android clock and timer app built with Kotlin and Jetpack Compose.

It is designed to turn an Android phone into a clean bedside or desk clock, especially in landscape mode. The app focuses on a dark OLED-friendly interface, simple timer controls, screen burn-in protection, and customizable clock styles.

## Features

- Fullscreen landscape clock
- OLED-friendly black background
- Keeps the screen awake while the app is open
- Burn-in protection with subtle clock shifting
- Quick timer presets: 5, 15, and 30 minutes
- Radial countdown timer UI
- Pause, resume, and reset timer
- Done screen when timer finishes
- Vibration feedback when timer ends
- Soft alarm sound when timer ends
- Battery saver warning
- Settings overlay
- Persistent settings using Android DataStore
- Dim mode for softer night colors
- 5 clock styles:
    - Classic
    - Stacked
    - Minimal
    - Split
    - Seconds
- Accent color picker for clock and timer UI

## Built With

- Kotlin
- Jetpack Compose
- Android DataStore
- Android vibration APIs
- Android audio APIs

## App Purpose

NightClock is meant for simple everyday use:

- Bedside clock
- Desk clock
- Nap timer
- Pomodoro-style timer
- Minimal night clock display

## Notes

For the best experience, Battery Saver should be turned off. Some Android devices may reduce vibration, sound, or background behavior when Battery Saver or Low Power Mode is enabled.
